업무 전체 흐름 파악하기

당장 급한 것들 간신히 틀어막으면서 조각조각 습득하는 단편적인 업무 지식을 전체 흐름으로 정리하는 과정이 필요합니다.

최종 표기일(로딩 기준): -

학교행정의 큰 덩어리

  • 회계 업무: 예산편성 → 집행 (세입/세출) → 정산
  • 공사·용역·물품 계약: 품의 → 계약(원인행위) → 이행 → 검수 → 지급
  • 인건비 지급: 대상자/근무정보 → 수당요건 → 산출 → 지급 → 기록
  • 복무·근태: 신청/결재 → 근태반영 → 증빙/기록
  • 기록물·문서: 접수/배부 → 시행/관리 → 보존/공개
  • 민감업무: 민원, 전력조회 등(리스크 관리가 핵심)

업무는 “프로세스(동사)”로 외운다

예시

계약 = 수요조사 → 품의(기안) → 계약체결 → 이행 → 검수 → 대금지급 → 기록보존

연가 = 신청 → 결재 → 근태반영 → 기록보존

업무별로 3가지만 확실히

  • Trigger: 언제 이 업무가 발생하나
  • 필수서류: 첨부·증빙·근거
  • 리스크: 틀리면 어디서 문제가 터지나(감사/민원/정산)
